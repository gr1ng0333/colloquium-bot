# Telegram-бот «Коллоквиум»

Бот хранит 33 билета в SQLite, принимает загрузку и редактирование через Telegram, собирает выбранные билеты в HTML с тёмной темой и KaTeX, а графики отправляет отдельными файлами.

## Установка

```bash
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
```

## Настройка

Создайте `.env` по шаблону `.env.example`:

```env
BOT_TOKEN=telegram_bot_token
ADMIN_ID=123456789
```

## Запуск

```bash
python bot.py
```

## Интерфейс

Пользователь видит кнопки:

```text
📋 Все билеты
❓ Помощь
```

Админ дополнительно видит:

```text
⚙️ Админка
```

В админке доступны загрузка билета, редактирование текста, управление графиком, удаление билета и статус загрузки.

## Пользовательский запрос

Отправьте номера билетов любым удобным способом:

```text
1, 22, 7
5 12 28
дай 22 билет
```
