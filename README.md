# Telegram-бот «Коллоквиум»

Бот хранит 33 билета в SQLite, принимает загрузку и редактирование через Telegram, собирает выбранные билеты в HTML с тёмной темой и KaTeX, а графики отправляет отдельными файлами. К одному билету можно прикрепить до 3 графиков.

## Установка

```bash
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
```

## Настройка

Создайте `.env` по шаблону `.env.example`:

```env
BOT_TOKEN=telegram_bot_token
ADMIN_ID=123456789
```

## Запуск

```bash
python bot.py
```

## Использование

Пользовательские команды:

```text
/start
/help
```

Примеры пользовательских запросов:

```text
1, 22, 7
5 12 28
дай 22 билет
```

Админские команды:

```text
/upload
/status
/delete 22
/cancel
```

Также доступны кнопки:

```text
📋 Все билеты
❓ Помощь
```

Админу дополнительно показывается кнопка:

```text
⚙️ Админка
```
